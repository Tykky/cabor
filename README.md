# Cabor C


## How to build

The build dependencies are
- CMake 3.22 or newer
- C99 compiler

to build on Windows use
```
build.bat
```
to build on Linux or macOS use
```
./build.sh
```
After building the program can be run on Windows with
```
cabor.bat
```
and on Linux or macOS with
```
./cabor.sh
```
